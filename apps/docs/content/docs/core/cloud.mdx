title: deployit Cloud
description: "Deploy your apps to multiple servers remotely without worrying about the underlying infrastructure."
---


deployit Cloud allows you to deploy your apps to a cloud provider of your choice. This means that you can deploy your apps to any cloud provider, such as AWS, GCP, Azure, or DigitalOcean, without having to worry about the underlying infrastructure.


By default when you install deployit in a Self Hosted version, if you deploy all your applications by default they will be deployed on the same server where deployit UI is installed. This means that you will need to build and run your applications where deployit 
UI is installed, which can be a challenge if you don't have the resources to do so, also self hosted support for remote instances.


deployit cloud starts from $4.50 per month per server, the next is 3.50$, and you can deploy as many applications you want to your remote server connected to a deployit cloud, multi server feature is recommended for HA and scalability projects.

You can start by registering on the [deployit Cloud](https://app.deployit.com) website and follow the steps to deploy your apps, see the [Pricing](/#pricing) page for more information.
