The deployit CLI can be used to create, deploy, and manage enviroments.

## Requirements

Is required to be already authenticated with the deployit CLI.


## Commands

1. `deployit env pull <file>` -  Pull environment variables from deployit in a file.
2. `deployit env push <file>` - Push environment variables to deployit from a file.
